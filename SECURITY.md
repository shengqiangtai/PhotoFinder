# Security Policy

## Reporting

Please report security issues privately through GitHub Security Advisories if the repository has them enabled. If advisories are unavailable, open an issue with a minimal description and avoid posting secrets, local database files, personal photos, or exploit details.

## Data Handling

PhotoFinder is designed to run locally. Photo paths, thumbnails, embeddings, and optional API keys are stored in the local application database under the user's application data directory. These files are ignored by Git and should never be attached to bug reports.

## Supported Versions

Security fixes target the latest source on the default branch until formal versioned releases are established.
