# Phase 4 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add usable search, image, and thumbnail APIs on top of the indexed photo vectors from Phase 3.

**Architecture:** Keep Phase 4 narrow. Add one `core/searcher.py` module that turns text into vectors and performs filtered vector ranking against SQLite, then expose that logic through a thin `api/routes/search.py`. Add `api/routes/image.py` for binary original-image and thumbnail responses, and extend schemas/app wiring only as needed for the new endpoints.

**Tech Stack:** FastAPI, aiosqlite, sqlite-vec, onnxruntime, Pillow, unittest.
